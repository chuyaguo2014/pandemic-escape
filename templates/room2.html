There are two ways to represent color: Additive and Subtractive.

With Additive Color, we start with the absence of color (Black), then add colors until we get the full spectrum of White. We can generate any color using the primary colors Red, Green, and Blue like so:

Green + Blue = Cyan

Red + Blue = Magenta

Red + Green = Yellow

Red + Green + Blue = White

Subtractive Color is the reverse of Additive color.

We start with the full color of White, then subtract colors to generate any color we want. If we subtract everything, we get Black as the absence of color like so:

White - Red - Green - Blue = Black
